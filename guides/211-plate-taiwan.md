# 211 餐盤外食版（211 Plate Method — Taiwan Dining Out Guide）

> **211 餐盤原則**：每一餐的盤子空間分配為：**蔬菜 2 份**、**蛋白質 1 份**、**澱粉 1 份**（體積比例 2:1:1）。  
> 不需要計算熱量，靠視覺比例控制每餐營養結構。

---

## 為什麼台灣外食需要特別說明

哈佛健康飲食餐盤（Harvard Healthy Eating Plate）和台灣衛福部 HPA 211 餐盤的原版示意圖，都是在家自行備餐的情境設計。但台灣外食族比例超過 70%，面臨的挑戰是：

- 便當盒的比例幾乎是「澱粉 1/2，蛋白質 1/4，蔬菜 1/4」——正好反過來
- 自助餐菜色的烹調多為大火油炒或滷汁重糖
- 超商套餐以便利為主，蔬菜攝取極低
- 夜市食物大多是炸物或重醬

這份指南提供每種台灣常見外食情境下的 211 落地方法。

---

## 211 餐盤基本規則（快速參考）

| 區塊 | 比例 | 份量參考 | 選擇原則 |
|------|------|---------|---------|
| 蔬菜 | 2/4（半盤） | 約 300–400g 生重 | 各色蔬菜，優先清炒或燙 |
| 蛋白質 | 1/4 | 掌心大小（約 100–150g） | 雞肉、魚、蛋、豆腐、豆類 |
| 全穀澱粉 | 1/4 | 一個拳頭大（約 150–200g 熟重） | 糙米 > 白米 > 精緻麵食 |
| 好油脂 | 少量 | 1–2 茶匙 | 烹調油、堅果、酪梨 |
| 飲料 | — | 無限量 | 白開水或無糖飲料 |

## 情境一：便當（排骨便當、雞腿便當、自助餐盒）

### 標準便當的問題

典型排骨便當（750kcal）的比例：
- 白米飯：約 250g（佔盒面積 50%）
- 主菜（排骨）：1 塊（約 80g）
- 配菜（通常 2–3 樣）：共 100–150g
- **蔬菜比例只有 15–20%，遠低於 211 的 50%**

### 便當的 211 調整策略

| 操作 | 方法 | 效果 |
|------|------|------|
| **減飯加菜** | 請便當店少飯，加一份燙青菜（通常 +$10–15）| 蔬菜比例從 20% → 35%+ |
| **雙蔬菜配菜** | 選 2 道蔬菜 + 1 道蛋白質主菜，不選炸物配菜 | 省熱量、補纖維 |
| **選雞腿 > 排骨** | 雞腿肉去皮蛋白質比例更高，炸豬排油脂多 | 蛋白質品質提升 |
| **搭配超商燙青菜** | 7-11 有燙花椰菜、菠菜，加到便當一起吃 | 快速補足蔬菜 |

### 建議便當組合

```
✅ 推薦：
  - 主菜：去皮雞腿 / 香煎雞胸 / 清蒸魚
  - 配菜：燙地瓜葉 + 炒高麗菜（都選蔬菜）
  - 主食：少飯（約 150g）或換地瓜
  - 估計熱量：550–650 kcal，蛋白質 30–35g

⚠ 需調整：
  - 主菜：炸排骨 + 滷蛋
  - 配菜：炒高麗菜 + 豆乾滷（豆乾較好，但搭一個就夠）
  - 主食：正常飯（250g）
  - 估計熱量：750–850 kcal
```

## 情境二：自助餐（台灣最接近 211 的外食場所）

自助餐是台灣外食族實踐 211 餐盤**最容易**的選項，因為可以自由選擇每樣菜的份量。

### 自助餐 211 選菜流程

**步驟 1：先拿菜（蔬菜 2 格）**
- 空心菜、地瓜葉、炒高麗菜、燙花椰菜、炒豆芽
- 目標：2–3 種蔬菜，共佔便當盒面積一半

**步驟 2：選蛋白質（1 格）**
- 優先：蒸魚、燉雞腿、豆腐、茶碗蒸、燙花枝
- 次選：滷豬肉（脂肪中等）、炒蛋
- 避免：炸排骨、炸雞塊（反式脂肪 + 裹粉額外澱粉）

**步驟 3：最後拿飯（1 格）**
- 視覺控制：拳頭大小，不要填平
- 若有糙米選糙米

**典型自助餐 211 套餐（約 $90–120）：**
| 菜色 | 份量 | 備註 |
|------|------|------|
| 糙米飯 | 1 碗（約 150g） | 最後拿，控量 |
| 滷雞腿（去皮） | 1 隻 | 主蛋白 |
| 燙地瓜葉 | 1 份 | 蔬菜 1 |
| 炒高麗菜 | 1 份 | 蔬菜 2 |
| 豆腐湯 | 1 碗 | 額外蛋白 |
| 估計合計 | ~550 kcal | 蛋白質 35–40g |

## 情境三：超商（7-11 / 全家 / 萊爾富）

超商最難達成 211，因為蔬菜選項少、比例低，需要刻意組合。

### 超商 211 組合策略

**核心問題**：超商單品蔬菜很少——御飯糰 + 雞胸肉 + 無糖豆漿是蛋白質組合，但沒有蔬菜。

**解決方案**：
1. 加購超商沙拉（全家/7-11 都有 $45–60 的生菜沙拉）
2. 加購燙花椰菜 / 玉米沙拉（7-11 鮮食區）
3. 自帶一包小番茄或芭樂

**超商 211 套餐範例（早餐）**：
| 品項 | 熱量 | 蛋白質 | 備註 |
|------|------|--------|------|
| 茶葉蛋 ×2 | 140 kcal | 16g | 蛋白質核心 |
| 御飯糰（鮭魚） | 190 kcal | 7g | 澱粉 |
| 生菜沙拉（無醬） | 30 kcal | 2g | 蔬菜（選無油醋醬） |
| 無糖豆漿 250ml | 45 kcal | 5g | 補充蛋白 |
| 合計 | ~405 kcal | ~30g | |

**超商 211 套餐範例（午餐）**：
| 品項 | 熱量 | 蛋白質 | 備註 |
|------|------|--------|------|
| 舒肥雞胸 | 100 kcal | 22g | 蛋白質核心 |
| 茶碗蒸 | 60 kcal | 6g | 補充蛋白 |
| 御飯糰（鮮蔬款）| 190 kcal | 5g | 澱粉 |
| 燙花椰菜 / 沙拉 | 40 kcal | 3g | 蔬菜 |
| 無糖綠茶 | 0 kcal | 0g | 飲料 |
| 合計 | ~390 kcal | ~36g | |

## 情境四：麵食館（牛肉麵、麵食小吃）

麵食館的 211 挑戰：澱粉（麵）比例天生偏高，蔬菜最少。

### 麵食館調整策略

| 操作 | 說明 |
|------|------|
| **點半份麵 + 加燙青菜** | 大部分麵館有燙青菜 $30–40，蔬菜比例直接翻倍 |
| **選清湯麵 > 乾麵** | 乾麵常有豬油拌，清湯熱量較可控 |
| **加滷蛋或豆腐** | 增加蛋白質，彌補麵食的澱粉主導 |
| **麵換蒟蒻麵** | 部分麵館提供蒟蒻麵選項（GI ≈ 0）|

**牛肉麵 211 調整範例**：
- 原版牛肉麵大碗：900 kcal，蔬菜 0–20g
- 調整後：半碗麵 + 燙小白菜 + 滷豆腐 = 650 kcal，蔬菜 150g+

## 情境五：火鍋（滿足 211 最容易的外食）

火鍋是台灣外食中**最容易達成 211** 的餐廳類型：
- 蔬菜選項多（高麗菜、菠菜、金針菇、豆腐、豆芽）
- 蛋白質選項多（雞肉片、豬肉片、魚片、蛤蜊、蛋）
- 主食份量自控（王道火鍋的飯後再點麵不是必要）

### 火鍋 211 選食清單

**大量補充（蔬菜區）**：
- 高麗菜、菠菜、娃娃菜、豆芽（低熱量，可大量）
- 金針菇、香菇（增加飽腹感）
- 豆腐、凍豆腐（植物性蛋白兼顧）

**適量選擇（蛋白質區）**：
- 雞肉片、豬里肌、魚片（各 1 份）
- 蛤蜊（高蛋白、低脂，優選）

**控量選擇（澱粉區）**：
- 玉米 1–2 節（低 GI）
- 芋頭幾塊（GI 55）
- 避免：王子麵（炸麵餅）、大量冬粉

**沾醬注意**：沙茶醬 1 湯匙 ≈ 80 kcal，無醬或少量醬油膏最佳。

## 211 外食快速評估法

外食前，用 3 個問題快速判斷：

1. **蔬菜夠嗎？** → 這餐有超過 1 份（約 100g）蔬菜嗎？沒有的話，想辦法補
2. **蛋白質夠嗎？** → 有一個掌心大小的蛋白質食物嗎（雞蛋/肉/豆腐）？
3. **澱粉是主角嗎？** → 白飯/麵/麵包佔超過一半？是的話減少份量

**3 題都過 → 這餐 OK**  
**2 題過 → 下一餐補充不足的部分**  
**1 題以下 → 認真調整，不要連續兩餐這樣吃**

## 常見 211 外食誤區

| 誤解 | 實情 |
|------|------|
| 「沙拉等於健康」| 乾島醬 / 凱薩醬 100ml = 300–500 kcal，等於一碗飯 |
| 「少吃澱粉就夠了」| 若蛋白質不夠，肌肉流失反而更快 |
| 「素食等於低熱量」| 糖醋排骨豆腐、麻婆豆腐、素炸物熱量都不低 |
| 「吃飽才有體力減重」| 211 餐盤每餐 450–650 kcal 已足夠，不需要吃到脹 |
