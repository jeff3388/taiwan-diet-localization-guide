# 超商健康點餐完整指南（7-11、全家、萊爾富）

> 台灣便利商店密度世界第一（每2,000人一家），是台灣人最高頻的外食場景。  
> 本指南幫助你在不改變習慣的情況下，做出更有益健康的選擇。

---

## 快速原則

1. **蛋白質優先**：每餐至少 20g 蛋白質，減少飢餓感
2. **看鈉含量**：單餐鈉 < 800mg 為佳
3. **避免含糖飲料**：改選無糖茶、無糖豆漿、黑咖啡
4. **御飯糰 > 三明治 > 麵包**（飽足感和蛋白質密度排序）
5. **加一顆茶葉蛋**：最低成本的蛋白質補充（~7g 蛋白質，$10）

## 場景化建議

### 早餐（上班前快速）

**預算 $60–80 / 熱量目標 ~400 kcal**

```
茶葉蛋 × 2         → 140 kcal, 14g 蛋白質
無糖豆漿 250ml     → 87 kcal,  7g 蛋白質
地瓜（中）1條      → 120 kcal
─────────────────────────────────────
合計：~347 kcal,  21g 蛋白質, 鈉 ~330mg
```

### 午餐（辦公室）

**預算 $120–150 / 熱量目標 ~550 kcal**

```
雞胸肉舒肥包       → 150 kcal, 24g 蛋白質
鮭魚御飯糰         → 185 kcal, 8g 蛋白質
生菜沙拉包（油醋）  → 45 kcal
無糖綠茶 600ml     → 0 kcal
─────────────────────────────────────
合計：~380 kcal,  32g 蛋白質, 鈉 ~700mg
```

### 下午點心（避免血糖崩潰）

**預算 $40–60 / 熱量目標 ~150 kcal**

```
無糖希臘優格 100g  → 100 kcal, 10g 蛋白質
核桃小包裝         → 90 kcal,  2g 蛋白質
─────────────────────────────────────
合計：~190 kcal,  12g 蛋白質
```
