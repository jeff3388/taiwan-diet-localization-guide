# DASH 飲食低鈉外食攻略（台灣版）

> Scientific basis: Appel et al. (1997) DASH Trial, NEJM  
> Target: 每日鈉攝取 < 2,300mg（標準版）或 < 1,500mg（嚴格版）

---

## 台灣飲食鈉含量現況

台灣外食鈉含量普遍偏高：

| 食物 | 鈉含量（mg） | DASH 每日上限比例 |
|------|------------|-----------------|
| 碗粿（一碗） | ~1,200 mg | 52% |
| 泡麵（一包） | ~1,800 mg | 78% |
| 滷肉飯（一碗） | ~900 mg | 39% |
| 排骨便當（一盒） | ~1,400 mg | 61% |
| 手搖飲（700ml 含糖） | ~150 mg | 7%（但糖分問題大） |
| 清炒蔬菜（一份） | ~300–500 mg | 13–22% |

**結論：一頓台式外食可能用掉 60–80% 的每日鈉配額。**

## DASH 飲食核心目標

| 營養素 | DASH 目標 | 台灣人均攝取 |
|--------|----------|------------|
| 鈉 | < 2,300mg/天 | ~4,100mg/天（國健署調查） |
| 鉀 | ≥ 4,700mg/天 | ~2,200mg/天 |
| 鎂 | ≥ 500mg/天 | ~260mg/天 |
| 鈣 | ≥ 1,250mg/天 | ~580mg/天 |
| 飽和脂肪 | < 6% 總熱量 | ~11% 總熱量 |

## 點餐實用原則

### 通用技巧（所有場景）

1. **要求少醬、少鹽** — 中式餐廳通常接受
2. **湯不要喝完** — 一碗清湯鈉含量 600–900mg
3. **醬料另放** — 沾醬自行控制，不要淋在食物上
4. **選「清蒸」「燙」「烤」，避免「紅燒」「滷」**
5. **不加醬油膏** — 改用薑絲、蔥、香菜增味

---

### 便當店 / 自助餐

**選法：**
- 主食：糙米飯（若有）半碗，白飯也可，控制份量
- 蛋白質：蒸魚 > 豆腐 > 雞胸肉（要求不加滷汁）
- 蔬菜：選清炒、水煮，避免酸菜、泡菜、醬瓜（高鈉醃製品）
- 不要附湯，或要求清湯（白開水兌湯）

---

### 麵食館 / 麵攤

| 選 | 鈉估算 | 避 | 鈉估算 |
|----|----|----|----|
| 乾麵（醬料少放）| ~600mg | 紅燒牛肉麵 | ~2,200mg |
| 湯麵不喝湯 | ~800mg | 豬骨湯底拉麵 | ~2,800mg |
| 陽春麵 + 燙青菜 | ~700mg | 麻辣湯底 | ~3,000mg+ |

---

### 超商 / 便利商店

**低鈉組合：**

| 食品 | 鈉含量 | 備註 |
|------|--------|------|
| 茶葉蛋 | ~200mg | 可接受 |
| 御飯糰（鮭魚）| ~380mg | 選鈉較低款 |
| 無糖豆漿 300ml | ~85mg | 鉀、鈣來源 |
| 生菜沙拉（不加醬）| ~30mg | 需搭配其他蛋白質 |
| 低鈉礦泉水 | 0mg | 多喝水有助鈉排出 |

## 高鉀食物清單（台灣容易取得）

DASH 飲食要求高鉀攝取以平衡鈉的升壓效果：

| 食材 | 鉀含量（100g） | 取得方式 |
|------|-------------|---------|
| 番薯葉 | 457mg | 自助餐、傳統市場 |
| 菠菜 | 466mg | 超市、傳統市場 |
| 香蕉 | 358mg | 超商、水果攤 |
| 酪梨 | 485mg | 全聯、家樂福 |
| 豆腐 | 121mg（一份 300g = 363mg） | 任何地方 |
| 鯖魚（生） | 502mg | 超市、傳統市場 |
| 鷹嘴豆 | 718mg | 全聯進口區 |
