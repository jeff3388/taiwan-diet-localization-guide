# 低 GI 飲食台灣版（Low-GI Diet Taiwan Adaptation）

> **GI（升糖指數）** 測量食物讓血糖上升的速度，以葡萄糖=100為基準。  
> **GL（升糖負荷）** = GI × 每份碳水公克數 / 100，同時考量攝取量，更接近實際影響。

---

## 核心概念

低 GI 飲食（GI ≤ 55）的主要好處：
- 避免血糖快速波動，減少飯後疲倦感
- 延長飽腹感，減少不必要零食衝動
- 適合血糖管理（糖前期、第二型糖尿病）
- 配合熱量赤字時，有助維持肌肉量

**重要**：GI 高不代表不能吃——西瓜 GI 76，但每份 GL 只有 4（水分占 92%）。同時看 GI 和 GL 才準確。

## 外食低 GI 實戰策略

### 便當 / 自助餐
1. **主食選擇**：糙米飯 > 白飯；地瓜替代白米（若有供應）
2. **份量控制**：一個拳頭大的飯（約 150g），其餘空間給蛋白質和蔬菜
3. **配菜選擇**：清炒蔬菜 > 糖醋類（加工糖分會提高 GL）
4. **湯品注意**：避免甜湯（南瓜湯加糖、玉米濃湯）

### 麵食館
- 選烏龍麵（GI 62）或白麵（GI 49）> 米線（GI 58）
- 指定「乾麵」而非「湯麵」：減少吸入高鈉湯底
- 加豆腐、蛋或肉：增加蛋白質，進一步延緩血糖上升

### 早餐店
| 選項 | GI 分析 | 建議 |
|------|---------|------|
| 蛋餅（原味） | 中等（麵皮 GI ~60） | 可以，加蛋更好 |
| 饅頭夾蛋 | 中等（精緻白麵）| 可接受 |
| 厚片吐司+果醬 | 高（白麵包 GI 70+，果醬高糖） | ⚠ |
| 燕麥粥 | 低（GI 55）| ✅ 首選 |
| 飯糰（糙米）| 低–中 | ✅ 推薦 |

## 降低白米飯 GI 的實用技巧

白米飯是台灣飲食核心，不需要完全避免——以下方式可降低 GI：

1. **冷卻再加熱**：煮好的白飯冷藏後，抗性澱粉增加，GI 約降 10–15%
2. **加醋**：少量蘋果醋或白醋（每餐 1–2 湯匙）可減緩消化速度
3. **混合雜糧**：白米中加 30% 糙米或燕麥，有效降低整體 GI
4. **搭配蛋白質先吃**：先吃蛋白質和蔬菜，最後吃飯，緩和血糖上升
5. **不要過度軟爛**：飯煮得太黏糊 GI 更高；保留一些嚼勁

## 一天低 GI 飲食範例（台灣版）

| 餐次 | 食物 | GI 分析 |
|------|------|---------|
| **早餐** | 大燕麥粥 + 無糖豆漿 + 茶葉蛋 | 低 GI，蛋白質充足 |
| **午餐** | 糙米飯 1 碗 + 烤雞腿 + 炒地瓜葉 + 蛤蜊湯 | 中–低 GI 組合 |
| **下午** | 芭樂半顆 或 無糖希臘優格 | 低 GI 點心 |
| **晚餐** | 地瓜（水煮）+ 豆腐炒蔬菜 + 毛豆 | 低 GI，植物性蛋白 |
| **飲品** | 無糖綠茶全天 | GI 0 |

估計全天 GL：45–55（建議每日 GL 目標 ≤ 80）
